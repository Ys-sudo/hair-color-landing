Multiple object detection using pre trained model in TensorFlow.js

Difficulty: Easy

How to use

Please wait for the model to load before trying the demos below at which point they will become visible when ready to use.

Demo: Webcam continuous classification

Hold some objects up close to your webcam to get a real-time classification! You must be on the https version of the website for this to work. When ready click "enable webcam" below and accept access to the webcam when the browser asks (check the top left of your window)

Please note: This demo loads our desired machine learning model via an easy to use JavaScript class made by the TensorFlow.js team to do the hard work for you. No machine learning knowledge is needed to use this. View the link to learn more about fine tuning this machine learning model. See our other tutorials if you want to load a model directly yourself, or recognize a custom object using your own data.

See the README.md or the project page for more information.
